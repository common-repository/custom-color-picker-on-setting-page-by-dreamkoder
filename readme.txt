This plugin will add Color Picker on setting page and selected color can be used using [dk-setting-color]

== Description ==
Sometimes you need admin to give ability to select color for navigation, body or some other element. So just select color on setting page and use use [dk-setting-color] in visual editors or do_shortcode("[dk-setting-color]") in PHP files.
